# PriceFinder
The user can put in a book from a fake website and their target price. If the real price meets the target price, they get sent an email.
--------------------------------------------------------
So my original idea was I would have the user put in a link to a target product, their email and then their ideal price. If and when the price would change to meet their target they would get an email notifiying them about the price change. Although I soon enough realised that target easily blocked my requests and instead turned to using a fake book website which is meant to be scraped.
